# Tooling

The format is specified in [Format specification](/format). This section
specifies the tool that operates on it: the CLI, its CI patterns, and the
VS Code extension.

{% cards %}
{% card title="CLI" href="/tooling/cli" icon="⌨️" %}
One binary, `docref`, with twelve commands for checking, refreshing, approving, and exploring references.
{% /card %}
{% card title="CI patterns" href="/tooling/ci-patterns" icon="🔁" %}
Gate every push, sync snippets mechanically, and watch cross-repo drift on a schedule.
{% /card %}
{% card title="VS Code extension" href="/tooling/vscode-extension" icon="🧩" %}
CodeLens, diagnostics, quick fixes, drift diffs, autocomplete, and a collection sidebar over the same core.
{% /card %}
{% /cards %}
